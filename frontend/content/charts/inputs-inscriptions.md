Shows the daily number of inputs that reveal an inscription.

<!--more-->

An input is counted when it reveals an [ordinals inscription](https://docs.ordinals.com/inscriptions.html) via a Taproot script-path spend.
A single transaction can contain multiple inscription-revealing inputs.
